# Transition & moment paintings (optional)

Drop public-domain Romantic-era paintings here and rebuild (`node build.js`).
Missing files fall back gracefully — screens use the parchment gradient and
event "moment" cards simply don't appear (the banner still announces them).
Optimise to ~1280px-wide JPEG (~150–250 KB) before committing; a dark scrim is
layered over each for legibility, so darker/medium reproductions read best.

| File | Where it shows | Suggested public-domain painting |
| --- | --- | --- |
| `title.jpg`   | Title screen           | Leutze, *Washington Crossing the Delaware* (1851) |
| `victory.jpg` | Game over — you won     | Trumbull, *Surrender of Lord Cornwallis* (1820) |
| `defeat.jpg`  | Game over — you lost    | Trumbull, *Death of General Warren at Bunker's Hill* (1786) |
| `france.jpg`  | Moment — France enters  | Couder, *Siege of Yorktown* (1836) |
| `capital.jpg` | Moment — a capital falls| Trumbull, *Death of General Mercer at Princeton* (1831) |
| `winter.jpg`  | Moment — a brutal winter| Trego, *The March to Valley Forge* (1883) |

All are firmly public domain (artists dead 150+ years, or US works pre-1928);
find them on Wikimedia Commons by the exact title.
